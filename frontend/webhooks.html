integration_instructions GitHub Webhook Setup Guide

  1. 1

    Start ngrok

    Run .\start-ngrok.ps1 in the project root. Copy the https:// forwarding URL.

  2. 2

    Open your GitHub repo → Settings → Webhooks → Add webhook

    Repeat for each repo. Use the Payload URL and Secret shown on this page.

  3. 3

    Configure the webhook

    • • Payload URL: copy from the field above
    • • Content type: application/json
    • • Secret: copy from the Webhook Secret field above
    • • Events: select Pushes and Pull requests
  4. 4

    Push a commit — watch it appear live

    Branch naming controls priority: hotfix/* → P1, main → P2, release/* → P3, develop → P4, feature/* → P5.
